  \item
    Пусть дано дерево $T = \langle V, E \rangle$. Для каждого ребра вычислить, сколько
    простых путей проходит через него. $\O(|V| + |E|)$.

    Решение:

    Каждое ребро разбивает граф на 2 компоненты связности. В первой компоненте выбираем, откуда начинаем, во второй - где заканчиваем.
    Получается, нужно перемножить размеры компонент связности.

    Надо для каждого ребра посчитать размер одной из компонент связности. Для этого подвесим дерево за одну из вершин.
    Выберем одно ребро. Одна из компонент дерева будет лежать в поддереве нижней вершины ребра. 

    Запускаем DFS на этом поддереве, получаем его размер $S$. Тогда размер оставшегося дерева равен $N - S$. Тогда через ребро проходит $S * (N - S)$ простых путей
